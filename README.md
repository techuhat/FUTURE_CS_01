# FUTURE_CS_01 — Web Application Security Testing

This repository contains the work completed for Task 1 of the Future Interns Cyber Security Internship.

## Objective

Perform vulnerability assessment of a web application using OWASP Top 10 methodology.

## Tools Used

- Kali Linux
- Burp Suite
- OWASP ZAP
- Nmap
- DVWA

## Vulnerabilities Identified

- SQL Injection
- Cross Site Scripting (XSS)
- Authentication Weakness
- Security Misconfiguration
